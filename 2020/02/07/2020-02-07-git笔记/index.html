一、git概述

  1. Git 是一个开源的分布式版本控制系统，是 Linus Torvalds为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

  2. git基本概念

    • 工作区
    • 暂存区
    • 本地仓库
    • 远程仓库

  3. GIT 与 svn 主要区别

    • 存储方式不一样：git按元数据存储，svn按文件存储
    • 使用方式不一样：命令不同,git需要三个命令来提交，SVN只需要commint来提交
    • 管理模式不一样：git是分布式的管理系统，SVN是一个远程集中式的管理系统

  4. 强烈建议所有的git仓库都有README, LICENSE, .gitignore三个文件
