hexo博客搭建

SuperSuperZ

2020-02-05

环境

windows10 64位
nodejs v12.14.0
npm 6.13.4

一、安装hexo

  1. 新建一个文件夹，用于存放博客的所有内容

  2. 在新文件夹下打开命令行

  3. 键入命令npm i -g hexo，安装hexo

  4. 查看hexo版本hexo -v

二、初始化

  1. 初始化
  2. 初始化完成后，当前文件夹内会多出以下文件或文件夹
    • node_modules：各种依赖包
    • public：存放生成的静态文件
    • scaffolds：一些通用的markdown模板
    • source：自己编写的markdown文件，_drafts草稿文件，_posts发布的文章
    • themes：主题
    • _config.yml：整个博客的配置
    • db.json：source解析所得到的
    • package.json：项目所需模块项目的配置信息

三、hexo常用命令

  1. 创建新文章

  2. 运行hexo服务器
    (PS：默认部署到本机4000端口，可直接访问预览效果）

  3. 生成静态文件

  4. 部署到远程网站

四、关联远程网站，搭建github pages（以github为例）

  1. 在github上面新建仓库(New repository)

  2. 设置仓库名(Repository name)为 yourname.github.io，其中yourname需要修改为你的github用户名，否则不能用于github pages

  3. 设置好后，创建仓库(Create repository)

  4. 生成 SSH Key，在用户文件夹下新建.ssh文件夹，并在命令行中运行一下命令

  5. 将生成的 id_rsa.pub 中的文件内容，粘贴到github中（个人 - 设置(setting) - SSH and GPG keys - New SSH key - Key文本框）。Title随便填，主要用于区分不同的SSH key。

  6. 测试SSH key是否添加成功，本地命令行输入下方命令

  7. 编辑hexo文件夹下的 _config.yml ,修改deploy的部分，其中repo记得换成你自己的仓库

  8. 安装hexo部署所需的git

  9. 部署,过程中需要输入github的用户名和密码

  10. 在浏览器输入 http://yourgithubname.github.io，即可查看你的博客了

五、更换主题

  1. 下载主题zip包或者，git clone一个主题到项目的 themes 文件夹下

  2. 修改项目根目录下的 _config.yml 文件中 theme的值为下载的主题名称即可

  3. 执行 hexo generate & hexo server，然后浏览器访问 “localhost:4000” 预览效果
