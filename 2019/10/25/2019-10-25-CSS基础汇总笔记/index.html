  1. <div>标记

    div标签的属性说明
    属性名 描述
    position定位方式 static 静态定位 (默认)
    absolute 绝对定位(与位置属性等结合使用可实现元素的绝对定位)
    relative 相对定位(对象不可层叠，但也要依据位置属性来设置元素的具体偏移位置)
    width、height宽高 width 定义层的宽度
    height 定义层的高度
    top、bottom、right、left设置位置 auto 默认值

    float浮动 (不仅仅用于图像和表格) left 表示浮动元素在左边，是居左对齐
    right 表示浮动元素在右边，是居右对齐

    clean清除 left 表示不允许在某元素的左边有浮动元素

  2. <span>标记
    相对于<div>,用于范围较小的地方，常用于行内元素。

  3. <div>和<span>区别

  • 相同点：<div>和<span>都没有对元素内的对象进行任何样式的定义，主要用于应用样式表，两者都可用来产生区域范围，以定义不用的文字段落。
  • 不同点：
    • 区域内是否换行： <div>标记区域内的对象与区域外的上下文间会自动换行，而<span>不会。
    • 标记相互包含： 当需要同时用到这两个标记是，建议用<div>包含<span>标记。否则有可能会造成<span>标记的区域不完整而形成断行的现象
    • 定位不同: <div>是块元素，指定定义HTML的容器；<span>是行内元素，指定内嵌文本容器
